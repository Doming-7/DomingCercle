## 📐 What It Does

Enter a radius value and instantly get the **surface area** of a circle using the formula:

```
A = π × r²
```

---

## 🚀 Features

- Enter any radius value
- Click **Calculer** to compute the area
- Instant result displayed on screen
- Lightweight — no frameworks, no dependencies beyond Bootstrap
